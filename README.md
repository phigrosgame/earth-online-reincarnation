# 🌌 輪迴模擬器 — 超凡之路

一款由 **AI 即時生成劇情**的網頁人生模擬遊戲。從一個再普通不過的人開始，一年一年活過你的一生；死亡之後，把這一世累積的代幣化作詞條，讓下一次輪迴走得更遠 —— 從凡人、富豪，一路到修仙、神明。

> 純前端單一 HTML 檔，**雙擊 `index.html` 即可遊玩**，不需安裝任何東西。劇情由 Google Gemini 生成（免費），無金鑰時也能進入但無法生成故事。

---

## 🎮 怎麼玩

1. 下載或開啟 [`index.html`](index.html)（直接用瀏覽器打開）。
2. 第一次進入會要求填入 **Gemini API Key**：到 [aistudio.google.com/apikey](https://aistudio.google.com/apikey) 免費取得，貼上即可（可貼多把，會自動輪替）。
3. 選擇世界、性別 → 從 0 歲開始你的一生。
4. 日常一年年推進，遇到**重大抉擇**時做選擇（或自己手動輸入行動）。
5. 死後結算獲得代幣 → 去**詞條抽取所**抽卡、背包裡裝備詞條 → 再次輪迴，變得更強。

## ✨ 核心玩法

### 隱藏命運數值
- **可見**的只有現實看得到的東西（如財富、名聲）。
- **健康、運氣、智慧、心性…** 全部隱藏，只能靠事件（健康檢查、考試、算命）偶爾窺見當下狀態。
- **健康**是壽命的隱藏計算器：過勞、生病會扣健康，歸零即死。

### 章節式 AI 生成
- 每個重大事件選完，一次生成「接下來幾年的日常小事 + 下一個重大事件」。
- 日常小事文字精簡、自動推進；部分附帶無關緊要的輕量選項，想互動就點。
- 重大事件提供 **4 個選項 + 1 個自由手動輸入**，選項不顯示數值，全憑情境判斷。

### 骰子決定成敗，AI 只負責描寫
- 創業、打官司、比試、賭一把等**成敗未定**的選擇，由程式依你的隱藏數值與運氣**擲骰**出四級結果（大成功／小成功／失敗／慘敗），再交給 AI 照結果生成劇情 —— AI 不能擅自把失敗寫成成功。

### 三種「非自然死亡」
- **純意外**（車禍、地震等天災人禍）：與行為無關，一生累積約 5%。
- **遭報復橫死**：得罪黑道、權貴、仇家會累積「威脅值」，日後可能被暗殺。
- **疾病病逝**：長期拚命拖垮健康，健康過低就有病死風險。

### 輪迴成長（Roguelike）
- 第一世就是個平凡人，**打破平凡、出人頭地非常困難**，需要長期積累、正確抉擇加上運氣。
- 越投胎越強：靠跨世累積的代幣與詞條，逐步逼近首富、權傾天下，乃至超凡。

## 🎴 詞條與代幣經濟

| 代幣 | 取得 | 用途 |
|---|---|---|
| **🔹 A 幣** | 每次人生結算 | 抽詞條、轉永久基礎數值 |
| **🔸 B 幣** | 精彩人生偶發 | 擴充詞條攜帶格、高階保底抽卡、解鎖新世界 |

- **詞條** 5 階稀有度：N / R / SR / SSR / UR，機率遞減（55 / 28 / 12 / 4 / 1%），十連保底 SR+。
- **合成升級**：同詞條集滿 5 張升一級（最高 Lv.5），加成遞增。
- **被動 + 觸發**：詞條既加隱藏數值/機率，也會寫進 AI 提示讓它生成專屬劇情（如「重生者」知曉未來、「金手指」綁定系統）。
- **攜帶格有限**：每世要取捨配裝，不會「玩一次就買完」。

## 🌍 世界

| 世界 | 解鎖 |
|---|---|
| 🌆 現代凡人 | 預設 |
| 💎 豪門世家 | 🔸 24 |
| ⚔️ 古代王朝 | 🔸 56 |
| 🌀 修仙境界 | 🔸 120 |
| ☀️ 神明境域 | 🔸 300 |
| 🌌 混世界 | 解鎖任意 3 個世界後，🔸 40 |

> **混世界**：投生前手動勾選要混入哪些已解鎖世界的設定（可加隨機詞條），讓現代、修仙、神明等題材交織出現。

## 🤖 AI 設定

- 預設模型：`gemini-3.1-flash-lite`。
- 支援**多把金鑰輪替**：某把撞到額度限流（429）會自動冷卻並換下一把，疊加免費額度。
- 金鑰只存在瀏覽器 `localStorage`，不會上傳任何伺服器。
- 遇到問題可開瀏覽器 **F12 → Console** 看詳細診斷（HTTP 狀態、API 錯誤、JSON 解析）。
